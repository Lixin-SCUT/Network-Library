## 项目简介
### 项目的简介
  参考了muduo数据库的思想，分析其中的差异，进行进一步的思考与学习，并实现一个基于Reactor的模式的网络库。

1. one thread one loop模式
2. RAII机制
3. EPOLL
4. 缓冲区Buffer设置
5. 如何优雅地关闭连接
6. log日志

### 项目目的
  Web服务器能够很好的贯穿之前所学的知识，包括之前看过的《C++ Primer》、《STL源码剖析》、《计算机网络》、《TCP/IP详解》《UNIX网络编程》等书 涵盖了下列知识点

1. TCP协议
2. 多进程多线程
3. I/O
4. 线程同步
5. C++语法
6. 编程规范
